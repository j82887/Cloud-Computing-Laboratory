## 01. 安裝Dlib
* 請使用pip安裝下列六個套件： numpy, scipy, matplotlib, scikit-learn, jupyter, opencv-python
* 再行安裝Dlib，有兩種安裝方式：
  * pip install dlib (不建議)
  * Dlib官網下載包 https://pypi.org/simple/dlib/ (下載dlib-19.8.1-cp36-cp36m-win_amd64.whl)
